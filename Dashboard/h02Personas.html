# 支持平台

  • 目前仅支持 Twitter 和 Facebook ，后续可能添加更多应用的添加。默认排序为 Twitter、Facebook。
    • 未关联社交账号显示链接按钮。
    • 已经关联社交账号显示相关内容。

# 工具栏

  • 无任何账户时候，默认进入 Personas 创建页面。
  • Toolbar 显示当前 Personas 账户，点击后下拉显示 Personas 管理面板。
  • 添加 Personas 验证长度，最高不能超过 24 位。超出后，错误显示：用户名超出最大长度
  • 删除 Personas 需要密码进行验证。
  • 备份账户，点击后跳转至设置页面，进行备份。
  • Toolbar 显示 Personas 账户下拉，显示所有 Personas 账户列表。显示当前 Personas 状态。单个 Personas 编辑名称，链接/解除链接 社交账户。默认每个 Personas 可以链接一个 Twiteer 和 Facebook 账户。

# 备份提示

  • 备份时间提醒，每过一个自然月提醒一次，删除后当月不显示提醒。点击后跳转至 Settings 中备份。

# 联系人

  • # 无好友

    • 显示空状态，引导用户分享下载链接。

  • # 有好友

    • 未安装 Mask 好友

      • 进行邀请，对应平台进行推文邀请@目标用户，进行推荐下载。
      • 删除好友

    • 已安装 Mask 好友

      • 显示分组 / 选择分组按钮
      • Favorite 功能，让用户进行列表优先显示，显示按字母顺序进行展示。
      • 点击选择分组，弹出显示当前分组列表。并显示当前用户的分组状态。用户无分组，列表显示选择分组，有分组显示最多 不超出分组最大显示宽度，超出显示...。
      • 分组弹出，显示当前所有分组，并优先显示已经加入的分组。剩余分组按字母进行排序。
        • 搜索，进行模糊搜索。有按字母进行结果排序，没有显示无搜索结果。
      • 删除好友，点击后直接将好友纳入已删除名单。屏蔽该用户查看本账户的发送的加密信息。一律为装有 Mask 无权限显示。

    • 已删除，显示已经删除的用户，并且可以还原联系人状态。

# 加密群组

  • # 无群组

    • 显示空状态，可以进行群组创建。

  • # 有群组

    • 新建群组：最高创建群组，200 个。超出列表不展示+按钮。创建名称长度不超过 24 个字符。超出长度显示：群组名称不能超出 24 个字符

    • 编辑群组：点击展示弹窗，显示所有好友列表。

      • 优先显示已添加用户，并显示按钮已添加。按钮划过显示删除。
      • 未添加用户显示添加按钮。
      • 搜索用户，模糊搜索用户名称和账号名称。无结果显示无结果状态。

    • 分组后，显示分组卡片，

  • 分组卡片，显示分类标题，分类用户的头像（最多显示 5 个）和分类数量。

    • 划过分类卡片显示：重命名、编辑、删除 按钮。

# 文件

目前使用的是 Arweave 文件系统。每次上传文件是需要一定的费用。目前大约 0.1AR 100mb。Arveave (opens new window)是一个致力于永久存储的分布式协议。目前所有的费用由 Mask 来进行支付。后续用户量持续扩张，超出费用使用的能覆盖的额度。需要能使用账户能调取 Arweave 插件进行支付。或者选择其他的去中心化存储服务方进行存储。旨在为用户提供可靠的去中心化文件存储。避免任何机构对用户进行数据隐私的收集。

目前市面上已有的去中心化存储服务方及介绍：

  • # IPFS

    Filecoin 是一个基于 IPFS 激励层上开发的分布式存储网络协议，宗旨为了建立人们可以使用存储重要信息的安全空间，打造一个基于区块链系统的数据存储和检索方法。它是协议实验室以星际文件系统为基础而创造的该项目由 Protocol Labs 主导开发，通过 PoST （复制证明）和 PoRep （存储证明）机制来确保用矿工已按要求存储文件，保证了文件的存储安全，持续生成证明矿工一直在存储该文件，确保文件的长期保存。

  • # Storj

    Storj 目前是一个基于以太坊开发文件存储解决方案应用。成立于它使用加密、文件分片和基于区块链的哈希表在 P2P 络上存储文件。其主要目的在于纯粹的存储，因为在以太坊上发行也就没有了特别的共识模式 2014 年完成首次区块链融资，并于 2017 众筹了 3000 万美金，项目立项为区块链驱动的去中心化版的 Dropbox。

  • # Arweave AR

    Arweave 项目于成立于 2017 年 4 月，主网在 2018 年 6 月上线，AR 类似于 Filecoin 的模式，也是 IPFS 协议中的一个激励层应用，是一种基于区块链和加密货币技术的新型存储方式。AR 其愿景是解决传统互联网中存储中心化、系统脆弱的问题，具有全球化，永久性，低成本，防篡改的特点。

  • # 上传

    • 上传文件，每个用户上传文件，每个文件不能超出 10.5MB 。每个 Personas 每天上传不能超出 50。超出后 Upload 按钮置灰。

  • # 创建文件夹

    • 创建文件夹上限 1000（待定）个。超出 Create new folder 按钮置灰。
    • 文件夹名称字数不超过 128 英文字符/64 个中文字符。超出后无法继续输入。

  • # 移动

    • 选中文件的时候出现 Move 按钮 ，显示弹框可以移动选中文件或者文件夹进行目录调整。
    • 移动弹出 Modal 中可以在当前文件夹中创建文件夹。

  • # 重命名

    • 单个选中文件时候显示 Rename 按钮。命名规则与创建文件夹名称长度一致。

  • # 删除

    • 删除文件不提示，直接放入垃圾箱。不删除文件，仅分类进入到垃圾箱中。

  • # 已删除

    • 已删除文件无自动删除功能，彻底删除并确认二次删除提示，删除存储的文件和信息。
      • 单条删除确认删除当前文件；
      • 多条删除显示删除文件数量

  • # 文件预览

  • # 搜索

    • 无搜索结果
    • 有搜索结果，模糊搜索，按搜索匹配度和字母顺序进行排序。列表展示 20 行，超出分页。

  • 文件管理系统，

    • 初始默认页面，侧边栏显显示，上传按钮，创建文件夹和 已删除文件夹。
    • 选择单个文件，侧边栏显示下载、移动、重命名、删除和已删除文件夹。
    • 选择多个文件，侧边栏显示下载、移动、删除、已删除文件夹。
    • 点击移动按钮，弹出 移动 Move to 模块。显示文件顶级目录。 可以在当前目录进行添加目录。
    • 搜索按钮，搜索文件名字，回车键后显示搜索结果。没有结果显示 No Results
    • 已删除文件，显示文件列表，侧边栏显示 还原和永久删除按钮。
