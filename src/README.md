## 安装

- 克隆项目至本地

``` shell
git clone git@github.com:xiazhiming/MaskPM.git
```

- 安装 vuepress

``` shell
yarn init // 初始化项目
yarn add -D vuepress // 安装vuepress
yarn add -D vuepress-plugin-auto-sidebar // 安装导航插件
npm i -g lint-md-cli // 安装语法检查器
```
